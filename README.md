# SUNA SHOP

SUNA Shop est une landing page e-commerce moderne réalisée dans le cadre de la rotation S5 d'Akieni Academy.

Le projet met en avant une boutique fictive avec un catalogue de produits, une fiche produit et une page de contact.

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Git
- GitHub

## Fonctionnalités

- Landing page
- Catalogue responsive
- Fiche produit
- Contact
- Responsive Design
